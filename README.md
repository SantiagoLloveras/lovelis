[lovelis](lovelis.vercel.app) es un proyecto web hecho íntegramente con IA usando Cursor, basado en React + Vite. El repositorio está en desarrollo e incompleto, pero sirve como muestra de exploración, aprendizaje y construcción asistida por IA.

[lovelis](lovelis.vercel.app) is a web project built entirely with AI using Cursor, based on React + Vite. The repository is under development and incomplete, but it serves as an example of AI-assisted exploration, learning, and construction.
